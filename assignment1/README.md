#Steps to download and run Assignment 1: Spam Detection
1. Clone this repository and pull from Github
2. Open the csci2020u_mk/assignment1/devAssignment1 folder in Intellij
3. Run 'Main" class
4. Select the csci2020u_mk/assignment1/data folder via the directory chooser and click 'open'
5. The program will run from there

6. csci2020u_mk/assignment1/secondaryInstructions.txt  describes ways in which the spam detector can be improved.
