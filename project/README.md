#Steps to download and run Final Project: Cold Mail
1.Clone this repository and pull from Github<br />
2.Open the csci2020u_mk/project/Assignment3Homepage folder in Intellij<br />
3.Run 'Main" class<br />
4.The program will run from there<br />

Cold Mail is an Email client application which utitlizes the JavaMail api.

It allows the user to:<br />
-Log into a pre-existing email by entering their username and password<br />
-Choose between GMail, Windows Live Hotmail, Outlook, Rogers and Yahoo Mail services<br />
-View all emails in the inbox via a listView<br />
-Select emails to read them in a textArea<br />
-Reply to any given email<br />
-Compose a new email, entering the recepient, subject and complete message<br />
-Use bold, italics, alignments and different font sizes<br />

Extras:<br />
-Checks for input error upon signing in<br />
-Uses threads for mail<br />
-Clean and easy to use interface with JavaFX's UI tools<br />
-Implemented with the JavaMail and JavaBeans Activation Framework APIs<br />
